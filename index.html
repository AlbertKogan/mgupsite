Media queries

Media queries являются одной из важнейших составляющих современных адаптивных сайтов. Зоопарк экранов, встроенных во всё что только можно, начиная с банальных смартфонов и ноутбуков и заканчивая холодильниками и всякими концептуальными вещами просто огромен. Такая ситуация ставит нас в достаточно затруднительное положение, а именно какой концепцией построения сайтов пользоваться: создавать под каждый тип устройств свою версию сайта или сделать одну универсальную версию для всех. У каждого подхода есть свои плюсы и минусы и при проектировании нужно исходить из ситуации. Но если выбираете адаптивность, то media queries станут вашими друзьями.

Итак, что такое media queries (далее медиа запросы)? Это одна из особенностей спецификации CSS3, позволяющая определять стили под конкретные устройства, экраны, разрешения и так далее. Запросы начинаются с правила @media, после чего следует конкретное объявление стилей. Например 

                        @media screen and (max-width: 480px){
                            background-color: red;
                        }
Здесь говорится, что если идёт отображение на экране монитора и ширина окна меньше 480px, то цвет фона будет красный.

Одна из особенностей медиа запросов состоит в том, что можно сочетать различные параметры с помощью логических операторов AND, NOT. Например: 

                        @media screen and print{
                            ...;
                        }
                                    
                        @media all and (not speech) and (min-width: 720px){
                            ...;
                        }

В первом примере задаём стили для экранов и принтеров. Во втором для всех устройств с минимальной шириной 720 px кроме речевых синтезаторов.

Также немаловажной частью медиа запросов являются медиа-функции, которые задают характеристики устройств на которых будет отображаться документ. Список часто встречающихся функций:

  1. aspect-ratio - определяет соотношение ширины и высоты отоброжаемого окна
  2. height - определяет высоту окна
  3. width - определяет ширину окна
  4. orientation - определяет ориентацию устройства в пространстве

Более подробно о медиа-функциях и медиа-запросах можно почитать здесь (англоязычная документация) и здесь (справочник на русском).

@font-face

С помощью правила @font-face мы можем загружать и настраивать шрифты, которые отсутствуют на компьютере пользователя, тем самым полёт вашей типографской фантазии не ограничен. Собственно само правило не ново, оно появилось ещё в спецификации CSS 2.0, но по каким-то непонятным причинам к версии 2.1 оно исчезло. Синтаксис этого правила: 

                        @font-face {
                            font-family: family-name;
                            src: uri;
                            unicode-range: urange;
                            font-variant: font-variant;
                            font-feature-settings: normal|feature-tag-value;
                            font-stretch: font-stretch;
                            font-weight: weight;
                            font-style: style;
                        }

Но есть небольшая проблема. Наверное вы замечали, особенно при использовании Chrome, что на нектороых сайтах шрифты выглядят как будто сошли с экранов игры 90 - х годов (пиксельный). Проблема заключается в различных способах рендеринга в различных операционных системах и броузерах. Об этом можно почитать в этой замечательной статье.

Transformation

  • Описание
  • Примеры

Свойство transform - одно из нововведений позволяющих манипулировать в пространстве различными объектами, поворачивать, маштабировать, искривлять. Поддерживается современными браузерами и IE 9. Задаётся таким синтаксисом: 

                            transform: <функция> [<функция>]* | none

  • Функция - функция трансформации
  • none - отменяет действие трансформации

Существуют несколько функций трансформации:

  • scale - масштабирование элемента по вертикали и горизонтали
  • skew - наклон элемента
  • rotate - поворот элемента
  • translate - сдвиг элемента по горизонтали или по вертикали
  • matrix - задаёт матрицу преобразования (подробнее здесь)
                                    #ex1{
                                    -webkit-transform: matrix(2, 0, 0.5, 1, 22, 1);
                                    }

Border radius

Border radius тоже является новым свойством, введённый в CSS3. Позволяет задавать радиус скругления для каждого угла. Является весьма популярным свойством, получтившим широкое распростронение.

border-radius: 15px 30px 45px 60px
border-radius: 25px
border-radius: 10em/5em
border-radius: 10em 5em/5em 10em

Box-shadow

Box shadow добавляет тень к элементу. Можно применить несеолько теней, указывая параметры через запятую. Если вы меняете радиус скругления у элемента он также меняется у тени. Можно задавыть следующие параметры:

  1. inset - внутренняя тень
  2. Сдвиг по горизонтали и вертикали (отрицательные значения сдвигают влево и вверх соответственно)
  3. Размытие. Задаём радиус размытия. Чем больше размытие тем светлее и больше становится тень
  4. Растяжение. Увеличивает размер тени
  5. Цвет
